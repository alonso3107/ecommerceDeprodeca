## Stack
- Nuxt 4 + Vue 3 + Tailwind v4, pnpm

## Direccion de Diseno
- Exaggerated Minimalism luxury e-commerce
- Sin border-radius
- Sin sombras
- Sin glassmorphism
- Sin blur

## Paleta
- negro `#1C1917`
- blanco `#FAFAF9`
- dorado `#A16207`
- stone `#D6D3D1`
- crema `#F5F0E8`

## Tipografia
- Rubik para headings (700-900)
- Nunito Sans para body (300-600)
- JetBrains Mono para labels
- Hero: `font-display text-[clamp(3rem,10vw,8rem)] font-black`

## Layout y Espaciado
- Espacio negativo generoso
- `py-20` para secciones hero
- `gap-8` entre cards
- `px-6` en mobile
- `lg:pl-16` en detalle de producto

## Iconografia y Copy
- Solo SVG geometricos minimalistas (circulo, linea, check cuadrado)
- Nunca emojis
- Codigo y UI copy en espanol

## Estructura
- Componentes en `~/features/<dominio>/components/`

## Imagenes
- Cards: `aspect-[5/6]`
- Detalle: `aspect-square`
- `object-cover`
- URLs Unsplash con `w=800&q=90`

## Git
- Commits convencionales sin emojis
